# 🎯 로또 (Lotto)

## 📌 구현 기능 목록
- [x] 구입 금액 입력 및 유효성 검증  
  - 1,000원 단위 확인  
  - 음수, 0, 문자열 입력 시 예외 처리  
- [x] 로또 발행  
  - 1~45 범위 내 중복되지 않는 6개 숫자 생성  
  - 구매 금액에 맞는 개수만큼 자동 발행  
- [x] 당첨 번호, 보너스 번호 입력 및 검증  
  - 1~45 범위 확인, 중복 금지  
- [x] 당첨 결과 계산  
  - 1~5등 판별 및 개수 계산  
- [x] 수익률 계산 및 출력 (소수점 첫째 자리 반올림)  
- [x] 예외 처리  
  - 모든 예외는 `[ERROR]`로 시작하는 메시지 출력  

## 🧪 테스트 항목
- 로또 번호 유효성 검사 (6개, 중복, 범위 초과 시 예외)  
- 자동 발행 기능 (`auto()`) 검증  
- 일치 개수 계산(`matchCount`) 및 보너스 번호 포함(`contains`) 검증
